Web Developer Applicants Test

This is a small code test that we would like you to perform to give us a feel for your coding.
We also plan on using it as a base for discussion during the interview.

Goal:

  • Make this data searchable using AJAX technology

Requirements:

  • Implementation must use Javascript, PHP and (X)HTML (all three)
  • Will be tested on Linux with Apache 2.2 and PHP 5.3
  • We will unpack your solution and run it from http://ourserver/applicants/yourname/

Meriting:

  • Writing code with reusability in mind
  • Search match precision
  • Solution elegance
  • Visual design

Keep in mind:

  • Frameworks are allowed, but not required
  • Comments can be useful for us to understand thought process
  • This exercise is not meant to take more then a few hours
